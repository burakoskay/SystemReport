## A Sudden Departure

Ghostty, a terminal emulator known for its customization capabilities and innovative features, is leaving GitHub. The reasons behind this move are not explicitly stated, but it comes after a series of notable developments.

## A Look Back at Ghostty's Features

Ghostty has been making waves in the tech community with its unique features. One of its standout capabilities is the ability to display images directly in the terminal. This feature has proven to be a game-changer for users who frequently work with diagrams and charts. For instance, users can generate SVGs on a server, convert them to PNGs, and display them inline, eliminating the need to SCP files to a laptop and open them in a browser.

The terminal emulator also boasts a custom-shader feature, allowing users to write GLSL fragment shaders that render behind terminal content. This feature has led to the creation of a collection of watercolor-inspired terminal backgrounds, each named after a real painting technique. The shaders can be easily added to a user's Ghostty config, and a script is available to randomize the shader and color each time a new terminal window is opened.

## Community Engagement and Alternative Projects

The departure from GitHub comes after a period of active community engagement. Users have been showcasing their projects built on top of Ghostty, such as a terminal-style AI coding agent running fully on Cloudflare Workers. This agent utilizes a SQLite-backed Durable Object and can expose files as real HTTP Worker endpoints.

Other projects have also been leveraging Ghostty's capabilities. For example, Honeymux, a TUI wrapper for tmux, offers a durable terminal workspace for agents, shells, and TUIs. Tmux-bar, a macOS menu bar app, provides one-tap switching between windows in a current tmux session.
