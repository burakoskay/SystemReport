## Issue with 'cat' Command
Using the `cat` command in iTerm2 is unsafe. This issue arises when displaying file contents, such as `readme.txt`. 
## Context
The original blog post can be found at https://blog.calif.io/p/mad-bugs-even-cat-readmetxt-is-not. Discussion on Hacker News has 76 comments and 138 points: https://news.ycombinator.com/item?id=47809190.
## Mitigation
According to the source, using `cat` in iTerm2 poses a safety risk. Users should refer to the original post for further information on this issue.
