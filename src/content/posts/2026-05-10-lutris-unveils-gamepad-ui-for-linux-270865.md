## A New Way to Game on Linux

Lutris has launched a gamepad-navigable frontend, providing a '10-foot UI' designed specifically for use with a controller. The application, available as a .AppImage file, allows users to browse and launch their Lutris library entirely with a gamepad. To use it, download the .AppImage from the Releases page, make the file executable, and run it. This new interface marks a significant shift towards making Linux gaming more accessible and user-friendly, especially for those who prefer the comfort of couch gaming.

## How it Works

The new UI is designed to be straightforward. Games are listed, and users can find games quickly. Game cards display cover art, playtime, and last played date. This visual approach helps users navigate their library more efficiently, leveraging the gamepad for a seamless experience. The focus on simplicity and ease of use aligns with Lutris's goal of broadening its appeal within the Linux gaming community.

## Features

The application also features a 'Now Playing' screen that appears while a game is running. A global shortcut allows users to toggle the UI's visibility without closing their game. This feature is particularly useful for gamers who want to keep an eye on their game status or quickly return to their game without navigating through multiple menus.
