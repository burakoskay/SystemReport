## Introduction

Hacker News, a popular link aggregator and message board for technologists, has launched an official API in partnership with Firebase, a cloud-based platform. The API provides near real-time access to data and enables developers to modernize their apps.

The API is available at https://hacker-news.firebaseio.com/ with initial documentation and examples written by Nick Sivo on the Hacker News Github account. Currently, it's read-only, but the team hopes to improve it over time and may later enable access to private per-user data using OAuth.

## Benefits and Features

The API was created to help developers who rely on scraping the site for data. By releasing a proper API, Hacker News aims to give everyone time to convert their code before launching new HTML. The partnership with Firebase provides well-designed, easy-to-use SDKs for iOS, Android, and the Web, in addition to a REST API for developers building server apps.

The API should make switching apps fairly painless, according to the team. The goal is to switch to a new rendering engine in three weeks, which will allow for faster iteration on the frontend and mobile optimization.

## Context and History

Hacker News was created in 2007 by Paul Graham, a venture capitalist and president of Y Combinator, as a side project to experiment with Arc, a new programming language. The site was initially called Startup News and focused on startup-centric conversations. However, Graham soon expanded the focus to include 'anything that good hackers would find interesting.'

The site has become a Silicon Valley institution, with many technologists checking it religiously. The community discusses various topics, including technical hiring, open-source projects, and academic papers.
