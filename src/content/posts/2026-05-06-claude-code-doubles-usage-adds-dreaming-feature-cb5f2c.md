## Anthropic's Claude Code Gets Significant Boosts

Anthropic has announced major updates to its Claude Code model, including doubled usage limits for most users and a new 'dreaming' feature for managed agents. These changes aim to enhance performance, flexibility, and functionality for users of the AI coding assistant.

## Concrete Details: What's Changing

The updates specifically target Claude Code, a component of Anthropic's Claude suite focused on coding and software development tasks. According to reports, Pro and Max users of Claude Code will see their usage limits doubled, from 5 hours to 10 hours. This increase is expected to allow users to run more extensive code reviews, complete larger projects, or work on complex tasks without hitting usage caps.

## Context and Implications

The doubling of usage limits for Claude Code follows a deal with SpaceX, indicating that Anthropic is actively seeking partnerships to expand its capabilities and reach. The company is also exploring the use of SpaceX's planned orbital data centers, which could further enhance the performance and scalability of its models.

## Technical Mechanics and User Benefits

For users, the increased limits mean more flexibility in working with Claude Code. Developers can now engage in longer sessions, explore more code, and receive more comprehensive feedback from the AI assistant. This is particularly beneficial for complex projects or when working with large codebases.
