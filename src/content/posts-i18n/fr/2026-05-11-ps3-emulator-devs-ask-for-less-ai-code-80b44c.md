## Les développeurs de l'émulateur PS3 submergés par le code généré par IA

Les développeurs d'un émulateur PlayStation 3 ont demandé poliment aux utilisateurs de cesser d'inonder leur projet de code généré par des outils d'intelligence artificielle. Cette demande intervient alors que le projet a du mal à gérer l'afflux de demandes de tirage.

L'émulateur, qui permet aux utilisateurs de jouer à des jeux PS3 sur d'autres appareils, est ouvert aux contributions de la communauté. Cependant, ces derniers jours, les développeurs ont été submergés par des demandes de tirage générées par des outils d'intelligence artificielle. Ces demandes contiennent souvent du code qui n'est pas pertinent pour le projet ou est de mauvaise qualité.

## Qu'est-ce qui se cache derrière cette demande ?

Dans un article sur Hacker News, les développeurs ont expliqué qu'ils apprécient l'enthousiasme de leur communauté mais doivent maintenir la qualité de leur projet. Ils ont déclaré qu'ils ne sont pas opposés au code généré par IA en principe mais doivent s'assurer que les contributions répondent à leurs normes.

Les développeurs ont également souligné que le code généré par IA reproduit souvent du travail existant ou n'est pas optimisé pour l'émulateur. Cela a entraîné une augmentation significative de la charge de travail pour les développeurs, qui doivent examiner et rejeter de nombreuses demandes de tirage.

## L'impact sur le projet

L'afflux de code généré par IA a provoqué de la frustration chez les développeurs, qui estiment que cela entrave leur capacité à se concentrer sur l'amélioration de l'émulateur. Ils ont demandé aux utilisateurs de s'abstenir de soumettre du code généré par IA et de se concentrer sur des contributions significatives au projet.

Les développeurs ont également fourni des directives sur la façon de contribuer au projet, notamment des conseils sur la façon d'écrire du code de haute qualité et de tester les modifications.
