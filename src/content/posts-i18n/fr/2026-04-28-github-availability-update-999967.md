## Histoire de GitHub

GitHub a été fondé en 2008 par Tom Preston-Werner, Chris Wanstrath et PJ Hyett. Depuis sa création, la plate-forme a connu une croissance rapide, devenant l'une des plus grandes communautés de développeurs au monde. En 2018, Microsoft a acquis GitHub pour 7,5 milliards de dollars, élargissant ainsi sa portée et ses ressources. L'acquisition a permis à GitHub de tirer parti de l'expertise de Microsoft dans le cloud computing et l'intelligence artificielle, améliorant ainsi sa plate-forme et ses services.
