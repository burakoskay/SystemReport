# OpenAI élargit son API avec l'intelligence vocale
OpenAI a ajouté l'intelligence vocale à son API, qui, selon la société, est pratique pour les systèmes de service client et a des applications dans l'éducation et les plateformes de création. Cette fonctionnalité permet aux développeurs d'intégrer des capacités vocales dans leurs applications, transformant potentiellement la façon dont les utilisateurs interagissent avec la technologie.

## L'API vocale cible plusieurs domaines
L'API vocale peut être utilisée dans les bots de service client, les outils éducatifs et les plateformes de création. OpenAI met en avant trois domaines ciblés : les systèmes de service client, les applications éducatives et les plateformes de création. Cette expansion dans l'intelligence vocale pourrait marquer un changement significatif dans la façon dont l'API d'OpenAI est utilisée dans diverses industries, en particulier dans les secteurs où les outils à activation vocale deviennent de plus en plus courants. Par exemple, les bots de support client à activation vocale pourraient devenir plus courants, permettant aux utilisateurs de résoudre rapidement les problèmes sans avoir à interagir avec un représentant humain.

## Protection Trusted Contact
OpenAI a introduit une fonctionnalité appelée Trusted Contact pour signaler les conversations qui pourraient indiquer un préjudice. La société élargit ses efforts pour protéger les utilisateurs de ChatGPT dans les cas où les conversations pourraient tourner vers un préjudice. La protection est une nouvelle couche de sécurité conçue pour identifier et atténuer les risques potentiels associés aux plateformes de chat pilotées par l'IA. Cette décision souligne l'engagement d'OpenAI en matière de sécurité des utilisateurs et démontre une prise de conscience des risques potentiels associés aux plateformes de chat pilotées par l'IA.

## Plugin Codex pour Chrome
OpenAI a publié un plugin Codex pour Chrome. L'extension injecte le moteur de génération de code de Codex dans n'importe quelle page Web qui contient un éditeur de code. Le plugin pourrait transformer Codex en plus qu'un simple outil d'aide à la programmation, en faisant un outil polyvalent pour les développeurs et les non-développeurs. En intégrant Codex à Chrome, OpenAI rend ses capacités de génération de code plus accessibles et conviviales, ce qui pourrait plaire à un public plus large.
