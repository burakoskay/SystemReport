## Die Realität der KI-Erkennung

Reality Defender, ein Unternehmen, das an Lösungen zur Erkennung manipulierten Medienmaterials arbeitet, bietet eine API namens RealAPI an. Diese API ermöglicht es Entwicklern, die Erkennung von Deepfakes auf Unternehmsebene in jede App oder Plattform zu integrieren. RealAPI ermöglicht es Entwicklern, Bild-, Audio- oder Videodateien zur Erkennung zu senden, indem sie eine einfache Methode oder API-Anfrage verwenden. Die API liefert dann strukturierte JSON-Antworten mit Manipulation-Wahrscheinlichkeitsscores und erklärbaren Indikatoren. Diese Technologie kann verwendet werden, um manipulierte Stimmen, Bilder oder Videos zu erkennen, die bei Identitätsdiebstahl, Phishing oder Social-Engineering-Angriffen verwendet werden. Zum Beispiel kann die Technologie von Reality Defender sozialen Medienplattformen helfen, Deepfakes zu identifizieren und zu entfernen, die zur Verbreitung von Fehlinformationen verwendet werden könnten.

## Die Technik hinter RealAPI

Die Technologie hinter RealAPI basiert auf Machine-Learning-Algorithmen, die subtile Änderungen in Audio-, Video- und Bilddateien erkennen können. Diese Algorithmen werden auf großen Datenmengen von manipulierten und authentischen Medien trainiert, um die Muster und Anomalien zu lernen, die zwischen beiden unterscheiden. Die Verwendung von erklärbaren Indikatoren bietet Entwicklern ein klares Verständnis dafür, warum eine bestimmte Datei als manipuliert gekennzeichnet wurde, um entsprechende Maßnahmen ergreifen zu können. Wenn ein Entwickler beispielsweise RealAPI zur Erkennung von Deepfakes verwendet, könnte die API ein Video als manipuliert kennzeichnen, weil sie Inkonsistenzen in den Audio- und Videospuren erkennt.
