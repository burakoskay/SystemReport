## Einführung

Hacker News, ein beliebter Link-Aggregator und Message-Board für Technologen, hat eine offizielle API in Partnerschaft mit Firebase, einer cloud-basierten Plattform, gestartet. Die API bietet nahezu Echtzeit-Zugriff auf Daten und ermöglicht Entwicklern, ihre Apps zu modernisieren.

Die API ist unter https://hacker-news.firebaseio.com/ verfügbar, mit initialer Dokumentation und Beispielen, die von Nick Sivo auf dem Hacker News Github-Konto geschrieben wurden. Derzeit ist sie nur lesbar, aber das Team hofft, sie im Laufe der Zeit zu verbessern und möglicherweise später den Zugriff auf private Daten pro Benutzer mithilfe von OAuth zu ermöglichen.

## Vorteile und Funktionen

Die API wurde erstellt, um Entwicklern zu helfen, die auf das Scraping der Website für Daten angewiesen sind. Durch die Veröffentlichung einer richtigen API möchte Hacker News allen die Zeit geben, ihren Code zu konvertieren, bevor eine neue HTML-Version gestartet wird. Die Partnerschaft mit Firebase bietet gut gestaltete, leicht zu verwendende SDKs für iOS, Android und das Web, zusätzlich zu einer REST-API für Entwickler, die Server-Apps erstellen.

Die API sollte den Wechsel zu neuen Apps ziemlich schmerzlos machen, so das Team. Das Ziel ist es, in drei Wochen zu einem neuen Rendering-Engine zu wechseln, der eine schnellere Iteration auf der Frontend- und mobilen Optimierung ermöglicht.

## Kontext und Geschichte

Hacker News wurde 2007 von Paul Graham, einem Risikokapitalgeber und Präsidenten von Y Combinator, als Nebenprojekt erstellt, um mit Arc, einer neuen Programmiersprache, zu experimentieren. Die Website wurde ursprünglich als Startup News bezeichnet und konzentrierte sich auf startup-zentrierte Gespräche. Graham erweiterte jedoch bald den Fokus, um 'alles, was gute Hacker interessant finden würden, einzubeziehen.'

Die Website ist zu einer Institution im Silicon Valley geworden, mit vielen Technikern, die sie regelmäßig überprüfen. Die Gemeinschaft diskutiert verschiedene Themen, einschließlich technischer Einstellungen, Open-Source-Projekten und akademischen Papieren.
