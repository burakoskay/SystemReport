## SOND 推出 AI 驱动睡眠耳塞

SOND，一家由 Bose 前睡眠产品负责人领导的睡眠科技初创公司，以 700 万美元的资金推出了 AI 驱动睡眠耳塞。该公司的产品旨在通过人工智能提高睡眠质量。SOND 的耳塞旨在提供更安静的睡眠体验。该资金轮和产品发布标志着睡眠科技行业的重大发展。

睡眠科技市场近年来增长显著，众多公司相继成立以解决睡眠质量问题。SOND 的 AI 驱动耳塞有望与市场上的现有玩家竞争。公司专注于使用 AI 提高睡眠质量，使其与其他睡眠科技公司区别开来。

SOND 的耳塞旨在跟踪睡眠模式并提供个性化建议以提高睡眠质量。该设备使用机器学习算法分析睡眠数据并提供睡眠模式洞察。这些信息可用于识别潜在睡眠障碍并提供有针对性的干预。

## Reality Defender 发布深度伪造检测 API

Reality Defender，一家 YC W22 初创公司，推出了深度伪造和 GenAI 检测 API。该 API 使开发者能够将企业级深度伪造检测集成到任何应用程序或平台中。Reality Defender API 可大规模检测操纵的图像、音频和视频。它提供具有操纵概率分数和可解释指标的结构化 JSON 响应。

近年来，深度伪造的兴起已成为一个重大问题，多个行业都受到操纵媒体的影响。Reality Defender 的 API 旨在通过提供强大的检测解决方案来解决这个问题。该 API 的大规模检测深度伪造能力使其成为希望保护自己免受操纵媒体负面影响的公司的宝贵工具。

Reality Defender 的 API 设计易于集成和使用。该 API 提供简单直观的界面，允许开发者快速轻松地将深度伪造检测添加到他们的应用程序中。该 API 还提供高精度，低误报率。

## Helply 推出 AI 支持代理

Helply，一家 AI 支持平台，推出了具有保证结果的 AI 支持代理。该平台每席位提供 0 美元，仅对 AI 结果收费。Helply 的 AI 代理加载完整的账户上下文，并在对完整、正确的答案有高度信心时解决工单。该平台已被多家公司采用，导致支持工单显著减少。

近年来，AI 在客户支持中的应用越来越流行，众多公司相继成立以解决支持工单解决的问题。Helply 的 AI 支持代理有望与市场上的现有玩家竞争。公司专注于提供保证结果，使其与其他 AI 支持平台区别开来。

Helply 的 AI 支持代理旨在为客户提供快速、准确的支持。代理使用机器学习算法分析客户数据并提供个性化支持。这种方法已被证明是有效的，许多公司看到支持工单显著减少。
