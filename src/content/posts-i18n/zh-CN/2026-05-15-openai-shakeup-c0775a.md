## OpenAI 的变革简介
OpenAI 正在重组其高管团队，Greg Brockman 将负责领导公司的产品。这一变化是 OpenAI 统一 ChatGPT 和 Codex 为一个核心产品体验的一部分。随着 OpenAI 致力于扩大其产品范围，包括推出个人理财版 ChatGPT，这一举措应运而生。

新的个人理财版 ChatGPT 允许用户连接其银行账户，并查看其投资组合表现、消费、订阅和即将到来的付款的仪表盘。此功能通过 OpenAI 与 Plaid 的集成实现，使公司能够从超过 12,000 家金融机构中提取信息。

## 安全问题和更新
OpenAI 也在通过更新其 Mac 应用程序（包括 ChatGPT 和 Codex）来解决安全问题。公司发现了与第三方开发工具 Axios 的安全问题，并正在采取措施保护其用户。虽然 OpenAI 没有发现任何用户数据被访问的证据，但它正在更新其安全认证以防止任何潜在风险。

更新将要求所有 macOS 用户将其 OpenAI 应用程序更新到最新版本。5 月 8 日后，旧版本的应用程序可能将不再工作。OpenAI 正在采取这一措施，以确保其用户的安全。

## 精神健康问题和 ChatGPT
OpenAI 还发布了对其 ChatGPT 用户在典型一周内可能表现出严重精神健康危机的粗略估计。根据公司的数据，约 0.07% 的活跃 ChatGPT 用户表现出与精神病或狂躁相关的可能精神健康紧急情况的迹象。此外，0.15% 的用户有涉及明显自杀计划或意图的对话。

OpenAI 已与超过 170 名精神科医生、心理学家和全科医生合作，以改善 ChatGPT 在涉及严重精神健康风险的对话中的回应。公司的目标是表达同情，同时避免确认没有现实依据的信念。

## 技术机制
个人理财版 ChatGPT 与 Plaid 的集成是一项重要的技术成就。Plaid 的 API 允许 ChatGPT 访问超过 12,000 家金融机构的财务数据，为用户提供对其财务状况的全面了解。此集成也是安全的，Plaid 使用加密和其他安全措施来保护用户数据。

OpenAI 的 Mac 应用程序的安全更新也是一个重要的技术发展。更新解决了与 OpenAI 使用的 Axios 开发工具的安全问题。通过更新其安全认证，OpenAI 正在采取主动措施保护其用户免受潜在安全风险。
