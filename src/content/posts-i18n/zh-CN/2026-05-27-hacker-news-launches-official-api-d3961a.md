## 介绍

Hacker News 是一个流行的链接聚合器和技术人员讨论板，近日与云平台 Firebase 合作推出了官方 API。该 API 提供近乎实时的访问权限，使开发者能够现代化他们的应用。

API 地址为 https://hacker-news.firebaseio.com/，初始文档和示例由 Nick Sivo 编写，托管在 Hacker News GitHub 账户上。目前，API 仅支持读取，但团队希望随着时间推移改进它，并可能通过 OAuth 启用私有用户数据访问。

## 优势和特点

该 API 的创建是为了帮助依赖抓取网站数据的开发者。通过发布正式的 API，Hacker News 旨在让所有人有时间在推出新 HTML 之前转换他们的代码。与 Firebase 的合作提供了设计良好、易于使用的 iOS、Android 和 Web SDK，以及用于构建服务器应用的 REST API。

根据团队的说法，API 应该使应用切换相当轻松。目标是在三周内切换到新的渲染引擎，这将允许前端和移动设备优化更快地迭代。

## 背景和历史

Hacker News 由风险投资家、Y Combinator 总裁保罗·格雷厄姆于 2007 年创建，作为一个 side project，用于测试新的编程语言 Arc。网站最初名为 Startup News，专注于初创企业相关的对话。然而，格雷厄姆很快将焦点扩大到包括 '任何好的黑客会感兴趣的东西'。

该网站已成为硅谷的机构，许多技术人员都对其非常关注。社区讨论了各种话题，包括技术招聘、开源项目和学术论文。

## 技术机制

Hacker News API 建立在 Firebase 的 Realtime Database 之上，提供了一个可扩展且安全的存储和检索数据的基础设施。API 使用 RESTful 架构，使开发者可以轻松地将其与应用程序集成。

API 目前提供对包括评论、帖子和用户信息在内的广泛数据的访问权限。团队计划随着时间的推移扩展 API 的功能，添加新的特性和数据源。
