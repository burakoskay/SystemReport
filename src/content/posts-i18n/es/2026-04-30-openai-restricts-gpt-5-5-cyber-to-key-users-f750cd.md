## OpenAI limita el acceso a GPT-5.5 Cyber
OpenAI lanzará su herramienta de pruebas de ciberseguridad, GPT-5.5 Cyber, solo para defensores cibernéticos críticos. La empresa anunció el acceso limitado en un breve comunicado.

La restricción limita el grupo de usuarios que pueden probar las fortalezas defensivas del modelo. OpenAI no ha divulgado los criterios para defensores cibernéticos críticos, dejando la definición vaga.

## Asociación con Yubico agrega protección de hardware
OpenAI lanzó una capa de seguridad opt-in para cuentas de ChatGPT que utiliza claves de seguridad de hardware de Yubico. Los usuarios que activan la función deben insertar una clave de Yubico o utilizar su capacidad NFC para completar la autenticación.

## Actualización de aplicaciones de Mac después de un problema de seguridad
OpenAI pidió a los usuarios de macOS que actualizaran sus aplicaciones de escritorio, ChatGPT y Codex, debido a un problema de seguridad con una herramienta de desarrollador de terceros, Axios. La actualización aborda el problema de seguridad y renueva las certificaciones de seguridad que verifican las aplicaciones como software legítimo de OpenAI.
