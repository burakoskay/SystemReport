## Microsoft mejora Windows Update con reversión automática de controladores

Microsoft ha introducido una nueva función para mejorar la experiencia de Windows Update en Windows 11. La función 'Recuperación de controladores iniciada en la nube' revierte automáticamente controladores defectuosos instalados a través de Windows Update. Esta actualización tiene como objetivo reducir la frustración asociada con Windows Update.

La nueva función reemplaza un controlador defectuoso con uno que funcionaba previamente a través de Windows Update. Actualmente, los usuarios deben revertir manualmente un controlador o esperar a que los proveedores de hardware publiquen uno nuevo. Esta actualización forma parte de los esfuerzos de Microsoft para solucionar problemas en Windows 11.

La función 'Recuperación de controladores iniciada en la nube' es una mejora significativa en el proceso de Windows Update. Al revertir automáticamente controladores defectuosos, Microsoft reduce la probabilidad de que los usuarios experimenten problemas con sus dispositivos. Esta función es especialmente útil para usuarios que dependen de sus dispositivos para tareas críticas.

## Apple corrige errores de Safari y amplía integración con el hogar

Apple ha lanzado Safari 26.5, que corrige errores de WebKit que podrían bloquear Safari o exponer datos de usuario. La actualización incluye una solución para una vulnerabilidad de WebKit que podría permitir que contenido web malicioso revelara información sensible del usuario.

Además, Apple Home ahora admite productos de aspiradoras y mopeds de Matic. La integración permite a los usuarios controlar dispositivos Matic utilizando Siri y Apple Home. Los productos de Matic Robots son conocidos por su calidad de construcción de nivel Apple y su enfoque en la privacidad y la inteligencia.
