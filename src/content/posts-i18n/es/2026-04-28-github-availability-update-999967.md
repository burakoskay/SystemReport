## Historia de GitHub

GitHub fue fundado en 2008 por Tom Preston-Werner, Chris Wanstrath y PJ Hyett. Desde su creación, la plataforma ha crecido rápidamente, convirtiéndose en una de las comunidades más grandes de desarrolladores del mundo. En 2018, Microsoft adquirió GitHub por $7.5 mil millones, expandiendo aún más su alcance y recursos. La adquisición ha permitido a GitHub aprovechar la experiencia de Microsoft en *cloud computing* e inteligencia artificial, mejorando su plataforma y servicios.
