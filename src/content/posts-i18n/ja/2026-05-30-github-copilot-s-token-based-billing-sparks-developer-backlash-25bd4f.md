## GitHub Copilotのトークンベースの課金モデルが開発者の反発を呼ぶ

MicrosoftのGitHub Copilotの黄金時代は終わりを迎えたようだ。GitHubのCopilotは、AIを利用したコーディングアシスタントで、トークンベースの課金モデルに変更され、開発者からの批判を浴びている。この変更は、以前の価格設定構造からの大きな転換点となっている。この新しいモデルでは、開発者は使用したトークンの数に基づいて課金されることになり、特にヘビーユーザーにとってはすぐに費用がかさむことになる。開発者たちは、この新しい価格設定モデルを「冗談だ」と呼んでいる。

トークンベースの課金モデルは、複雑なシステムを利用して、開発者が使用したトークンの数を追跡し、課金する。トークンは、Copilotによって処理されたコードの単位を表す。開発者がCopilotを使ってコードを書いたり修正したりすると、トークンを消費することになる。このモデルは、特にヘビーユーザーにとって、ツールのアクセス性や価値についての懸念を引き起こしている。
