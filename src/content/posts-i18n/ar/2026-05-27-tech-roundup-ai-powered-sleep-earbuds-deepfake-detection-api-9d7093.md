## ظهور SOND مع سماعات أذن ذكية للنوم تعمل بالذكاء الاصطناعي

شركة ناشئة في مجال تكنولوجيا النوم بقيادة رئيس منتجات النوم السابق في Bose، ظهرت SOND من السرية مع 7 ملايين دولار في التمويل لسماعات الأذن الذكية للنوم التي تعمل بالذكاء الاصطناعي. يهدف منتج الشركة إلى تحسين جودة النوم باستخدام الذكاء الاصطناعي. تم تصميم سماعات الأذن من SOND لتوفير تجربة نوم أكثر راحة. يمثل جولة التمويل وإطلاق المنتج تطورًا كبيرًا في صناعة تكنولوجيا النوم.

## Reality Defender تكشف عن اكتشاف Deepfake API

Reality Defender، شركة ناشئة في YC W22، أطلقت API لاكتشاف Deepfake و GenAI. يتيح API للمطورين إضافة اكتشاف Deepfake ذو الجودة العالية إلى أي تطبيق أو منصة. يكتشف Reality Defender API الصور والصوت والفيديو المعدلة على نطاق واسع. يوفر استجابات JSON منظمة مع درجات احتمالية التلاعب ومؤشرات قابلة للتفسير.

## Helply تقدم وكلاء دعم الذكاء الاصطناعي

Helply، منصة دعم الذكاء الاصطناعي، قدمت وكلاء دعم الذكاء الاصطناعي مع نتائج مضمونة. توفر المنصة 0 دولار لكل مقعد وتفرض رسومًا فقط على نتائج الذكاء الاصطناعي. يحمل وكيل Helply الذكاء الاصطناعي سياق الحساب الكامل ويحل التذاكر عندما يكون لديه ثقة عالية في إجابة كاملة وصحيحة. تم اعتماد المنصة من قبل العديد من الشركات، مما أدى إلى تخفيضات كبيرة في تذاكر الدعم.
