## Uma Nova Maneira de Jogar no Linux

A Lutris lançou uma interface navegável com gamepad, fornecendo uma 'interface de 10 pés' projetada especificamente para uso com um controlador. O aplicativo, disponível como um arquivo .AppImage, permite que os usuários naveguem e lancem sua biblioteca de Lutris inteiramente com um gamepad. Para usá-lo, baixe o .AppImage da página de lançamentos, torne o arquivo executável e execute-o. Essa nova interface marca uma mudança significativa em direção a tornar os jogos no Linux mais acessíveis e fáceis de usar, especialmente para aqueles que preferem o conforto dos jogos no sofá.

## Como Funciona

A nova interface é projetada para ser simples. Os jogos são listados e os usuários podem encontrar jogos rapidamente. As cartas de jogo exibem arte de capa, tempo de jogo e data de última jogada. Essa abordagem visual ajuda os usuários a navegar por sua biblioteca de forma mais eficiente, aproveitando o gamepad para uma experiência perfeita. O foco na simplicidade e facilidade de uso está alinhado com o objetivo da Lutris de ampliar seu apelo dentro da comunidade de jogos no Linux.

## Recursos

O aplicativo também apresenta uma tela 'Agora jogando' que aparece enquanto um jogo está sendo executado. Um atalho global permite que os usuários alternem a visibilidade da interface sem fechar o jogo. Esse recurso é particularmente útil para jogadores que desejam manter um olho no status do jogo ou retornar rapidamente ao jogo sem navegar por vários menus.
