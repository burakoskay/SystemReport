## Introdução

O Hacker News, um agregador de links e quadro de mensagens popular para tecnólogos, lançou uma API oficial em parceria com o Firebase, uma plataforma baseada em nuvem. A API fornece acesso em tempo real aos dados e permite que os desenvolvedores modernizem seus aplicativos.

A API está disponível em https://hacker-news.firebaseio.com/ com documentação inicial e exemplos escritos por Nick Sivo no repositório GitHub do Hacker News. Atualmente, ela é somente leitura, mas a equipe espera melhorá-la ao longo do tempo e pode permitir acesso a dados privados por usuário usando OAuth.

## Benefícios e Recursos

A API foi criada para ajudar os desenvolvedores que dependem de *scraping* do site para obter dados. Ao lançar uma API adequada, o Hacker News visa dar a todos tempo para converter seu código antes de lançar um novo HTML. A parceria com o Firebase fornece SDKs bem projetados e fáceis de usar para iOS, Android e Web, além de uma API REST para desenvolvedores que criam aplicativos de servidor.

A API deve tornar a troca de aplicativos bastante indolor, de acordo com a equipe. O objetivo é mudar para um novo mecanismo de renderização em três semanas, o que permitirá uma iteração mais rápida no frontend e otimização móvel.

## Contexto e História

O Hacker News foi criado em 2007 por Paul Graham, um capitalista de risco e presidente da Y Combinator, como um projeto paralelo para experimentar com Arc, uma nova linguagem de programação. O site foi inicialmente chamado de Startup News e focado em conversas centradas em startups. No entanto, Graham logo expandiu o foco para incluir 'qualquer coisa que bons hackers achariam interessante.'

O site se tornou uma instituição do Vale do Silício, com muitos tecnólogos verificando-o religiosamente. A comunidade discute vários tópicos, incluindo contratação técnica, projetos de código aberto e artigos acadêmicos.

## Mecânica Técnica

A API do Hacker News é construída sobre o Banco de Dados em Tempo Real do Firebase, que fornece uma infraestrutura escalável e segura para armazenar e recuperar dados. A API usa uma arquitetura RESTful, tornando-a fácil para os desenvolvedores integrarem com seus aplicativos.

A API atualmente fornece acesso a uma ampla gama de dados, incluindo comentários, posts e informações de usuário. A equipe planeja expandir as capacidades da API ao longo do tempo, adicionando novos recursos e fontes de dados.
