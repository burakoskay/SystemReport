# System Report

**Tech news and analysis** — editorial coverage of AI, hardware, software, gaming, and policy.

Live: [systemreport.net](https://www.systemreport.net)

---

## About

System Report is an independent tech publication delivering in-depth news coverage, expert commentary, and free browser-based privacy tools. Content is available in 11 languages.

## Features

- **Editorial coverage** across five beats: AI, hardware, software, gaming, and tech policy
- **Audio narration** on every article
- **Privacy tools** — client-side utilities (EXIF metadata stripper) that never upload data
- **Multilingual** — 11 languages with full hreflang support
- **AMP Web Stories** for visual summaries
- **RSS, JSON Feed, Podcast RSS** for syndication

## Local Development

```bash
npm install
npm run dev
```

Requires Node.js 22+.

## Build

```bash
npm run build
```

Output: `dist/` — static HTML deployed to Cloudflare Pages.
